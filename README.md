A premium restaurant homepage built with React, Vite, and Tailwind CSS.

## Features
- **Modern & Chic Design**: Dark mode interface with neon amber and teal accents.
- **Responsive UI**: Fully responsive layout for mobile and desktop.
- **Interactive Menu**: Expandable "Chef's Specials" section.
- **Glassmorphism**: Trendy glass effects on navbar and cards.

## Tech Stack
- React
- Vite
- Tailwind CSS

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/pravejansari/restaurent-app.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
